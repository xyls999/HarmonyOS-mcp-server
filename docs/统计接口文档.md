# A9 智慧家居后端 v5 · 统计接口文档

> `GET /api/stats` — 12类真实状态数据  
> 设备未接入不写入模拟数据，离线字段为 `null`

---

## 统计总览

| # | 类别 | 字段名 | 数据源 | 在线时 | 离线时 |
|---|------|--------|--------|--------|--------|
| 1 | 设备在线率 | `device_online_rate` | 全部设备 | `{total, online, offline, rate}` | `rate=0.0` |
| 2 | 区域连通状态 | `area_connectivity` | 4块板子 | `{online, ip, last_seen}` | `online=false, last_seen=null` |
| 3 | 客厅温度 | `living_temperature` | TEMP QUERY → temp | `{value, unit, is_alert}` | `value=null` |
| 4 | 客厅湿度 | `living_humidity` | TEMP QUERY → humi | `{value, unit, is_alert}` | `value=null` |
| 5 | 厨房烟雾 | `kitchen_smoke` | CMD4 → smoke_level, smoke_alarm | `{smoke_level, smoke_alarm}` | 全部null |
| 6 | 厨房热敏 | `kitchen_thermal` | CMD4 → thermal_mv, temp_alarm | `{thermal_mv, temp_alarm}` | 全部null |
| 7 | 厨房综合报警 | `kitchen_alarm` | CMD4 → alarm | `{alarm}` | `null` |
| 8 | 卫生间风扇 | `bathroom_fan` | CMD6 → motor_* | `{motor_running, motor_speed, motor_direction}` | 全部null |
| 9 | 卧室窗帘 | `bedroom_curtain` | CMD9 → curtain_* | `{position, moving, homed, close_limit, open_limit}` | 全部null |
| 10 | 所有灯亮度 | `all_lights` | 各区域灯状态 | `{brightness, is_on}` | 全部null |
| 11 | 报警联动状态 | `alarm_linkage` | 联动状态 | `{last_kitchen_alarm, buzzer_triggered, events_24h}` | `buzzer_triggered=null` |
| 12 | 轮询配置 | 顶层字段 | 内存 | `poll_interval_kitchen=1.0, poll_interval_other=10.0` | 始终有值 |

## 返回格式详解

### 1. 设备在线率

```json
"device_online_rate": {
    "total": 10,       // 总设备数
    "online": 3,       // 在线数
    "offline": 7,      // 离线数
    "rate": 30.0       // 在线率%
}
```

### 2. 区域连通状态

```json
"area_connectivity": {
    "living_room": {
        "name": "客厅",           // 区域中文名
        "ip": "192.168.1.62",    // 板子IP
        "port": 8000,            // 板子端口
        "online": true,          // 是否在线
        "last_seen": "2026-07-11T12:00:00"  // 最后在线时间，离线=null
    },
    "kitchen":     { "name": "厨房",   "ip": "192.168.1.23", "port": 8000, "online": false, "last_seen": null },
    "bathroom":    { "name": "卫生间", "ip": "192.168.1.63", "port": 8000, "online": false, "last_seen": null },
    "bedroom":     { "name": "卧室",   "ip": "192.168.1.64", "port": 8000, "online": false, "last_seen": null }
}
```

### 3. 客厅温度 (TEMP QUERY → `temp=X,humi=Y`)

```json
"living_temperature": {
    "value": 24.5,      // 温度值，离线=null
    "unit": "°C",
    "online": true,     // 设备是否在线
    "is_alert": false,  // 是否超阈值(>28°C)
    "threshold": 28     // 告警阈值
}
```

### 4. 客厅湿度 (TEMP QUERY → `temp=X,humi=Y`)

```json
"living_humidity": {
    "value": 52.0,      // 湿度值，离线=null
    "unit": "%RH",
    "online": true,
    "is_alert": false,  // 是否超阈值(>70%)
    "threshold": 70
}
```

### 5. 厨房烟雾 (CMD4 → content[1]=smoke_level, content[2]=smoke_alarm)

```json
"kitchen_smoke": {
    "smoke_level": 0,       // GP03电平，离线=null
    "smoke_alarm": false,   // 是否烟雾报警(低电平报警)，离线=null
    "online": true,
    "description": "GP03低电平报警"  // 在线时有描述，离线=null
}
```

**说明**: `smoke_level=0` 表示GP03低电平，`smoke_alarm=1` 表示烟雾报警(低电平报警逻辑)

### 6. 厨房热敏 (CMD4 → content[3]=temp_alarm, content[7:9]=thermal_mv)

```json
"kitchen_thermal": {
    "thermal_mv": 1650,    // GP07/ADC0毫伏值，离线=null
    "temp_alarm": false,   // 是否过热报警(ADC≤1400mV)，离线=null
    "online": true,
    "threshold_mv": 1400,  // 报警阈值
    "description": "ADC≤1400mV报警"  // 在线时有描述
}
```

**说明**: `temp_alarm=1` 表示热敏ADC电压 ≤ 1400mV

### 7. 厨房综合报警 (CMD4 → content[4]=alarm)

```json
"kitchen_alarm": {
    "alarm": 0,            // 1=烟雾或过热任一报警，离线=null
    "online": true,
    "description": "1=烟雾或过热任一报警"  // 在线时有描述
}
```

**联动规则**: 
- `alarm` 从 `0→1` (上升沿) → 向客厅发送 `BEEP ALARM`
- `alarm` 从 `1→0` (下降沿) → 发送 `BEEP OFF`
- 同一持续报警不重复触发

### 8. 卫生间风扇 (CMD6 → content[2:5]=motor_*)

```json
"bathroom_fan": {
    "motor_running": true,      // 是否运行，离线=null
    "motor_speed": 80,         // PWM速度0-100，离线=null
    "motor_direction": 1,      // 0=停止 1=正转 2=反转，离线=null
    "online": true,
    "direction_label": "正转"  // 中文方向，离线=null
}
```

**硬件**: TB6612驱动N20电机，GP01/PWMA, GP02/AIN1, GP04/AIN2, STBY=3.3V

### 9. 卧室窗帘 (CMD9 → content[2:8])

```json
"bedroom_curtain": {
    "position": 75,        // 估算位置0-100%，离线=null
    "moving": false,       // 是否正在步进，离线=null
    "homed": true,         // 是否完成限位定位，离线=null
    "close_limit": 0,      // 关闭限位开关状态，离线=null
    "open_limit": 0,       // 打开限位开关状态，离线=null
    "online": true
}
```

**硬件**: 28BYJ-48 + ULN2003，GP01→IN1, GP02→IN2, GP04→IN3, GP05→IN4, GP06→关闭限位, GP07→打开限位  
⚠️ `last_error=1` 表示回零超最大步数，电机未接入前不要执行 `curtain home`

### 10. 所有灯亮度汇总

```json
"all_lights": {
    "light_01": {
        "name": "客厅主灯",     // 设备中文名
        "room": "客厅",         // 所在房间
        "brightness": 80,      // 亮度0-100，离线=null
        "is_on": true,         // 是否开启，离线=null
        "online": true         // 设备是否在线
    },
    "light_05": { "name": "客厅氛围灯", "room": "客厅",   "brightness": null, "is_on": null, "online": false },
    "light_02": { "name": "厨房灯",     "room": "厨房",   "brightness": null, "is_on": null, "online": false },
    "light_04": { "name": "卫生间灯",   "room": "卫生间", "brightness": null, "is_on": null, "online": false },
    "light_03": { "name": "卧室灯",     "room": "卧室",   "brightness": null, "is_on": null, "online": false }
}
```

### 11. 报警联动状态

```json
"alarm_linkage": {
    "last_kitchen_alarm": 0,       // 上次厨房alarm值 (0=正常, 1=报警)
    "buzzer_triggered": null,      // 客厅蜂鸣器当前状态，客厅离线=null
    "udp_listening": true,         // UDP 8001监听是否成功启动
    "events_24h": 2,              // 最近24小时报警联动事件次数
    "rule": "alarm 0→1: BEEP ALARM / alarm 1→0: BEEP OFF / 不重复触发"
}
```

### 12. 轮询配置 (顶层字段)

```json
{
    "timestamp": "2026-07-11T12:36:45",
    "poll_interval_kitchen": 1.0,   // 厨房轮询间隔(秒)
    "poll_interval_other": 10.0,    // 其他区域轮询间隔(秒)
    ...
}
```

## 设备在线示例

当卧室(192.168.1.64)在线时，`bedroom_curtain` 返回:

```json
"bedroom_curtain": {
    "position": 50,
    "moving": false,
    "homed": true,
    "close_limit": 0,
    "open_limit": 0,
    "online": true
}
```

当卧室离线时:

```json
"bedroom_curtain": {
    "position": null,
    "moving": null,
    "homed": null,
    "close_limit": null,
    "open_limit": null,
    "online": false
}
```

**没有任何模拟/默认数据**，离线就是null。

## 其他相关接口

|------|------|
| `GET /api/check` | 连通检查+语音播报(哪些在线/离线) |
| `GET /api/devices` | 10台设备详情(离线=null) |
| `GET /api/sensors` | 5个传感器详情(离线=current:null) |
| `GET /api/hardware/status` | 4区域连通性 |
| `GET /api/alerts` | 活跃告警列表 |
| `GET /api/sensors/history` | 传感器历史(24h) |
| `GET /health` | `{v:5, hardware:true}` |
