# A9 App v1.1.0 Alignment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 HarmonyOS 手机 App 的认证、设备、记录和 AI 功能完整对齐 A9 App 接口 v1.1.0，并保持克制的竖屏战术线框视觉。

**Architecture:** `Http` 是唯一网络边界，负责 JWT、刷新、签名、超时和错误语义；业务 API 只负责请求与响应归一化；四个页面只消费业务类型并呈现独立加载/错误状态。现有页面结构保留，不恢复监测页和场景快捷按钮。

**Tech Stack:** HarmonyOS NEXT SDK 24、ArkTS、ArkUI、NetworkKit、Crypto Architecture Kit、Hypium/Hvigor、Python 静态契约测试。

## Global Constraints

- Base URL 为 `http://yuanzhe.tech`，业务入口只使用 `/api/app/*`。
- App 用户使用 JWT，不创建、不保存、不发送 D6 API Key。
- AI 对话读取超时至少 120 秒。
- 不显示在线率、场景快捷按钮、远程密钥管理或内部配置入口。
- 不回退或覆盖工作区中已有修改。
